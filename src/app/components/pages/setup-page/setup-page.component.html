Setting Up the Project

Follow this step-by-step guide to install and configure the Angular documentation web app on your local system. This guide covers installation, dependencies, and project structure.

2. Install Angular CLI globally by running:

npm install -g @angular/cli

Check if Angular CLI is installed correctly:

ng version

3. Cloning the Project

To get the latest version of the documentation web app, clone the repository using Git:

git clone https://github.com/your-repo/angular-docs.git

Navigate into the project directory:

cd angular-docs

4. Installing Dependencies

Once inside the project folder, install all dependencies by running:

npm install

This command will download and install all required **Node modules**.

5. Running the Application

After installing dependencies, start the Angular development server:

ng serve

Once the server is running, open a browser and go to:

http://localhost:4200
